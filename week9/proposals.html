Pokedex with PokeAPI

Purpose

The purpose of this application is to share with Pokemon fans a long list of their favorite characters, with information on health, attack, speed, etc.

Personal Purpose

In last week's activity, I saw the pokeAPI and it caught my attention, I have also played and enjoyed pokemon. I saw that with this API you can create this type of Pokedex, that is, a user interface that allows you to see different types of pokemon and their characteristics.

Audience

The audience is traditional and new Pokemon fans. This application allows parents and children to enjoy their favorite video game saga.

Data sources

The API to use is PokeAPI, from there it extracts the information to use.
